
### 安装

```
npm install
```

根目录下的server 文件为node 服务端

```
cd server

npm install

npm start
```

运行起来


然后跑 npm run serve
项目


### 数据库文件

服务端运行起来之后  自动生成表，可在navicat 可视化工具进行看

权限分布为：

0. 游客
1. 博主
2. 草鸡管理员


草鸡管理源可以进入管理后台进行查看博客数据
